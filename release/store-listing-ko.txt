Threads Clipper for Obsidian은 관심 있는 Threads 게시글을 ♡ 좋아요 또는 북마크하는 순간 Obsidian 볼트에 구조화된 마크다운 노트로 저장합니다. 복사·붙여넣기, 수동 정리가 필요 없습니다.


주요 기능

💾  좋아요/북마크 시 자동 저장 — 별도 클릭 없이 threads.net 또는 threads.com에서 ♡나 북마크를 누르는 순간 노트가 생성됩니다.

📝  YAML frontmatter가 포함된 깔끔한 마크다운 — 작성자, 게시글 URL, 시각, 태그, 주제, 댓글 수, 스레드 길이가 Obsidian 속성으로 저장되어 검색·그래프 기능을 그대로 활용할 수 있습니다.

🧵  스레드·리포스트·인용·캐러셀 지원 — 멀티 포스트 스레드의 순서가 보존되며, 리포스트와 인용은 frontmatter에 명시됩니다.

🖼️  이미지 처리 — 원본 CDN URL을 그대로 사용하거나, 볼트로 로컬 다운로드 (REST 모드 한정).

💬  댓글 수집 (선택) — 게시글 답글을 별도 섹션으로 수집. 작성자 답글만 / 최소 글자 수 필터 제공.

🤖  AI 변환 (선택) — 본인 API 키로 게시글을 구조화된 노트(핵심 요약, 주요 개념, 상세 분석, 실행 아이템, Feynman 설명)로 변환합니다.

🌐  한국어 / English UI — 브라우저 언어 자동 인식, 또는 옵션 페이지에서 직접 선택. 기본 프롬프트 템플릿도 선택한 언어를 따릅니다.


두 가지 저장 방식

• Obsidian URI (기본값, 플러그인 불필요)
  클립보드를 통해 obsidian://new URI로 노트를 생성하고, 마지막으로 활성화된 Obsidian 볼트에 저장합니다. 설치가 가장 간단하며, 이미지는 URL 참조로 보존됩니다.

• Local REST API
  Obsidian Local REST API 커뮤니티 플러그인이 필요합니다. 볼트에 직접 쓰기, 이미지 로컬 다운로드, 상세 에러 보고를 지원합니다.

옵션 페이지에서 언제든 모드를 전환할 수 있습니다.


⚠️  중요 — 가능하면 게시글 상세 페이지에서 저장하세요

Threads 피드 뷰는 메인 게시글의 DOM만 노출하기 때문에, 연결된 글타래(thread)와 답글(댓글)은 피드에서 수집되지 않습니다. 스레드 전체와 댓글을 한 노트에 담으려면 다음 순서를 권장합니다:

  1. 게시글을 클릭해 상세 페이지(URL이 /post/...)를 엽니다.
  2. 거기에서 ♡ 또는 북마크합니다.
  3. 확장 프로그램이 스레드의 모든 연결글을 캡처하고, 댓글 수집이 켜져 있으면 화면에 보이는 답글도 함께 가져옵니다.

피드에서 좋아요/북마크해도 메인 게시글은 정상 저장되지만, 글타래의 나머지와 댓글은 누락됩니다. 가능한 한 상세 페이지에서 진행해 주세요.


AI 변환 (선택)

기본값은 꺼짐. 활성화하고 본인 API 키로 프로바이더를 설정하면 저장되는 모든 노트가 다음 구조로 변환됩니다:

  1. 핵심 요약 (Executive Summary)
  2. 주요 개념 (테이블)
  3. 상세 노트
  4. 실행 아이템
  5. Feynman 설명
  6. 원문 (하단 보존)

지원 프로바이더: OpenAI, Google Gemini, Anthropic Claude, xAI Grok, Z.AI GLM, OpenAI 호환 Custom/Local 엔드포인트 (Ollama, LM Studio 등 — 로컬 엔드포인트는 API 키 생략 가능). 프롬프트 템플릿은 자유롭게 편집할 수 있습니다.


프라이버시

• API 키, 설정, 통계 등 모든 데이터는 Chrome 로컬 스토리지에만 저장됩니다. 외부로 나가는 통신은 아래 경우뿐입니다:
    – localhost의 Obsidian REST 플러그인 호출, 또는 URI 모드에서 로컬 클립보드 사용.
    – AI 변환이 켜져 있을 때, 본인이 입력한 API 키로 본인이 선택한 AI 프로바이더 호출.

• 분석, 텔레메트리, 제3자 트래커 없음.

• 확장 프로그램은 threads.net과 threads.com에서만 동작합니다.


필수 조건

• Chrome 116 이상.
• 로컬 Obsidian 볼트.
• (REST 모드 한정) Obsidian Local REST API 커뮤니티 플러그인 활성화 및 실행.
• (AI 모드 한정) 선택한 프로바이더의 API 키.
